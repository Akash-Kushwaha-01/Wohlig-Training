1.REST API - JSON

GET /users - HTML Document Render. - DONE
GET /api/user -  List all User JSON - DONE.
GET /users/1 - Get the user with ID 1  ( /1 is called Dynamic Path Parameter/ how to denote dynamic path parameter: GET/api/users/:id  (:id -> Variable |  Dynamic))
GET /users/2 - Get the user with ID 2 -DONE

POST /user - create new user
PATCH /users/1 - Edit the user with ID 1
DELETE /users/1 - Delete the user with ID 1
